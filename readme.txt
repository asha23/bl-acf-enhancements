Add drag and drop functionality to flexible content. Add the ability to add <strong>NEW!</strong> to content block items.

== Description ==

This plugin enhances the Advanced Custom Fields (ACF) plugin by adding drag and drop functionality to flexible content and allowing you to mark content block items as <strong>NEW!</strong>.
